# 예찬쓰 생일파티 초대장

2022년 6월 4일 예찬쓰 생일파티 초대장 웹사이트입니다.

## 기능

- 생일파티 일정 및 장소 안내
- 참석자 정보 수집 양식
- 기념품 제작을 위한 사진 및 정보 입력
- 지도 서비스 연동 (네이버 지도, 카카오 지도)

## 개발 환경 설정

### 설치
```bash
yarn install
```

### 개발 서버 실행
```bash
yarn start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하여 확인할 수 있습니다.

### 빌드
```bash
yarn build
```

## 기술 스택

- React 18
- SweetAlert2 (알림창)
- CSS3

## 배포

빌드된 파일은 `build` 폴더에 생성되며, 정적 호스팅 서비스에 배포할 수 있습니다.
